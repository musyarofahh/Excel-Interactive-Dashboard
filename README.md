# Excel-Interactive-Dashboard
This project demonstrates how to build an **interactive dashboard** using Microsoft Excel.   The dashboard allows users to explore data dynamically through **PivotTables, PivotCharts, and Slicers**, making data analysis more efficient and visually engaging.

## 🎯 Features
- Interactive charts (bar, line, pie).
- Dynamic filtering with Slicers and Timelines.
- Automated data summaries using PivotTables.
- Clean and user-friendly layout.
- Applicable for financial reports, sales tracking, or operational data.

## 🛠️ Tools & Technologies
- Microsoft Excel (2016 or later recommended).
- PivotTable & PivotChart.
- Slicer & Timeline.
- Conditional Formatting.

## 🚀 How to Use
1. Open the Excel file (`Interactive_Dashboard.xlsx`).
2. Navigate to the **Dashboard** sheet.
3. Use the Slicers/Timelines to filter data interactively.
4. Explore charts and summaries to gain insights.

## 📂 Project Structure
- `Data/` → Raw dataset used for analysis.
- `Dashboard/` → Main interactive dashboard.
- `README.md` → Documentation of the project.

## 📈 Example Use Cases
- Sales performance tracking.
- Financial reporting.
- Employee productivity monitoring.
- Academic or research data visualization.
